## Overview
This project demonstrates automation of:
1. A user registration web form using JUnit and Selenium
2. Web scraping of a stock price table and writing the data to a text file

## Technology
- Java (JDK 17)
- Gradle
- Selenium WebDriver
- JUnit 5
- ChromeDriver
